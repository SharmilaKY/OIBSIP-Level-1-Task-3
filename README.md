Temperature Converter

This is a simple temperature converter web application built using HTML, CSS, and JavaScript.
It allows users to convert values between Celsius, Fahrenheit, and Kelvin.

How to Use

Enter a temperature value.

Select the unit (Celsius, Fahrenheit, or Kelvin).

Click the Convert button to see the result.

Features

Clean and simple UI

Converts between all three temperature scales

Fully responsive design

Files

index.html — Main structure

style.css — Styling

script.js — Conversion logic

Run the Project

Just open index.html in any browser.
